## API Key Authentication

Authenticate requests using secure API keys, ideal for server-to-server and automated integrations.

### How to Use API Key Authentication

<Steps>
  <Step title="Obtain Your API Key">
    Register your project on the AltrumAI platform and generate an API key.
  </Step>
  <Step title="Add API Key to Request Headers">
    Include the <code>Project-API-Key</code> header in all API requests.
  </Step>
  <Step title="Test Authentication">
    Send a test request to verify authentication is working.
  </Step>
</Steps>

### Example Usage

```python title="api_key_auth_example.py"
import requests

url = "https://api.altrum.ai/v1/openai/chat/completions"
headers = {"Project-API-Key": "YOUR_API_KEY"}
data = {
    "model": "gpt-3.5-turbo",
    "messages": [{"role": "user", "content": "Hello!"}]
}
response = requests.post(url, headers=headers, json=data)
print(response.json())
```

<Accordion title="API Key Management Best Practices">
  <ul>
    <li>Rotate API keys regularly</li>
    <li>Never share API keys publicly</li>
    <li>Restrict API key permissions to the minimum required</li>
    <li>Monitor usage and revoke compromised keys immediately</li>
  </ul>
</Accordion>
