## What is the AI Gateway?

The **AI Gateway** provides a unified, secure, and policy-driven interface to leading AI providers and advanced enterprise features.

### Key Features
- Unified API compatible with OpenAI SDK
- Multi-model support with advanced guardrails
- Multiple authentication methods
- Advanced features: auto-caching, streaming, tool/function calling, and more
