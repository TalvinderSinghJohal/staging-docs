## Google AI via AI Gateway

Integrate Google AI models using the unified API, with built-in policy enforcement and monitoring.

### How to Use

<Steps>
  <Step title="Obtain API Key">
    Register your project and get your API key from the AltrumAI platform.
  </Step>
  <Step title="Send a Request">
    Use the following code example to send a prompt.
  </Step>
</Steps>

<CodeGroup>
  <CodeBlock language="python" filename="google_ai_example.py">
    import requests

    url = "https://api.altrum.ai/v1/google-ai/completions"
    headers = {
        "Project-API-Key": "YOUR_API_KEY"
    }
    data = {
        "model": "google-palm",
        "prompt": "Hello, world!"
    }
    response = requests.post(url, headers=headers, json=data)
    print(response.json())
  </CodeBlock>
  <CodeBlock language="javascript" filename="google_ai_example.js">
    const axios = require('axios');

    const response = await axios.post(
      'https://api.altrum.ai/v1/google-ai/completions',
      {
        model: 'google-palm',
        prompt: 'Hello, world!'
      },
      { headers: { 'Project-API-Key': 'YOUR_API_KEY' } }
    );
    console.log(response.data);
  </CodeBlock>
</CodeGroup>

### Rate Limiting

- **Standard:** 60 requests/minute per API key
- **Burst:** 120 requests/minute for up to 10 seconds
