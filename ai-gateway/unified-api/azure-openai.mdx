## Azure OpenAI via AI Gateway

Access Azure OpenAI models through the AI Gateway, with unified API compatibility and advanced security features.

### How to Use

<Steps>
  <Step title="Obtain API Key">
    Register your project and get your API key from the AltrumAI platform.
  </Step>
  <Step title="Send a Request">
    Use the following code example to send a prompt.
  </Step>
</Steps>

<CodeGroup>
  <CodeBlock language="python" filename="azure_openai_example.py">
    import requests

    url = "https://api.altrum.ai/v1/azure-openai/chat/completions"
    headers = {
        "Project-API-Key": "YOUR_API_KEY"
    }
    data = {
        "deployment_id": "my-deployment",
        "messages": [{"role": "user", "content": "Hello, world!"}]
    }
    response = requests.post(url, headers=headers, json=data)
    print(response.json())
  </CodeBlock>
  <CodeBlock language="javascript" filename="azure_openai_example.js">
    const axios = require('axios');

    const response = await axios.post(
      'https://api.altrum.ai/v1/azure-openai/chat/completions',
      {
        deployment_id: 'my-deployment',
        messages: [{ role: 'user', content: 'Hello, world!' }]
      },
      { headers: { 'Project-API-Key': 'YOUR_API_KEY' } }
    );
    console.log(response.data);
  </CodeBlock>
</CodeGroup>

### API Parameters

  The deployment ID of your Azure OpenAI resource.
</ParamField>
<ParamField name="messages" type="array" required>
  Array of message objects for the conversation.

### Rate Limiting

- **Standard:** 60 requests/minute per API key
- **Burst:** 120 requests/minute for up to 10 seconds
