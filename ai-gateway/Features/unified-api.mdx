## OpenAI via AI Gateway

Access OpenAI models through a unified, OpenAI SDK-compatible endpoint.

### Example: Calling OpenAI via AI Gateway using the OpenAI SDK

```python title="openai_example.py"
from openai import OpenAI

client = OpenAI(
    api_key="your-openai-api-key",
    base_url="https://your-gateway-url/v1",
    default_headers={
        "x-provider-name": "openai",
        "x-altrumai-key": "your-gateway-api-key"
    }
)

response = client.chat.completions.create(
    model="gpt-4",
    messages=[
        {"role": "user", "content": "What is the capital of France?"}
    ]
)

print(response.choices[0].message.content)
```
